# list-group-react-component 
A react component that displays a group of bootstrap list items and other related components

#Usage
1. Clone Repo
   ```bash
   git clone https://github.com/nuwagalb/list-group-react-component.git

   ```
2. Install modules
   ```bash
   npm install

   ```
3. Run app
   ```bash
   npm run dev

   ```
